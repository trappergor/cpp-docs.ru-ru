# <a name="movsq"></a>__movsq
**Блок, относящийся только к системам Microsoft**  
  
 Создает повторяющиеся строки перемещения (`rep movsq`) инструкции.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
void __movsq(   
   unsigned char* Dest,   
   unsigned char* Source,   
   size_t Count   
);  
```  
  
#### <a name="parameters"></a>Параметры  
 [выходной] `Dest`  
 Целевой для операции.  
  
 [in] `Source`  
 Источник операции.  
  
 [in] `Count`  
 Число учетверенных слова для копирования.  
  
## <a name="requirements"></a>Требования  
  
|Встроенная функция|Архитектура|  
|---------------|------------------|  
|`__movsq`|X64|  
  
 **Файл заголовка** \<intrin.h >  
  
## <a name="remarks"></a>Примечания  
 В результате первый `Count` учетверенных слова, на которые указывают `Source` копируются `Dest` строка.  
  
 Эта процедура доступна только как встроенная функция.  
  
## <a name="example"></a>Пример  
  
```  
// movsq.cpp  
// processor: x64  
#include <stdio.h>  
#include <intrin.h>  
  
#pragma intrinsic(__movsq)  
  
int main()  
{  
    unsigned __int64 a1[10];  
    unsigned __int64 a2[10] = {950, 850, 750, 650, 550, 450, 350, 250,  
                               150, 50};  
    __movsq(a1, a2, 10);  
  
    for (int i = 0; i < 10; i++)  
       printf_s("%d ", a1[i]);  
    printf_s("\n");  
}  
```  
  
```Output  
950 850 750 650 550 450 350 250 150 50   
```
