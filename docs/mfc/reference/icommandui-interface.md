# <a name="icommandui-interface"></a>Интерфейс Икоммандуи

Управляет командами пользовательского интерфейса.

## <a name="syntax"></a>Синтаксис

```
interface class ICommandUI
```

## <a name="members"></a>Участники

### <a name="public-methods"></a>Открытые методы

|name|Описание|
|----------|-----------------|
|[icommandui__Check](#check)|Задает для элемента пользовательского интерфейса для этой команды соответствующее состояние проверки.|
|[Икоммандуи:: Континуераутинг](#continuerouting)|Указывает механизму маршрутизации команд продолжить маршрутизацию текущего сообщения в цепочку обработчиков.|
|[Икоммандуи:: Enabled](#enabled)|Включает или отключает элемент пользовательского интерфейса для этой команды.|
|[Икоммандуи:: ID](#id)|Возвращает идентификатор объекта пользовательского интерфейса, представленного `ICommandUI` объектом.|
|[Икоммандуи:: index](#index)|Возвращает индекс объекта пользовательского интерфейса, представленного `ICommandUI` объектом.|
|[Икоммандуи:: Radio](#radio)|Задает для элемента пользовательского интерфейса для этой команды соответствующее состояние проверки.|
|[Икоммандуи:: Text](#text)|Задает текст элемента пользовательского интерфейса для этой команды.|

### <a name="remarks"></a>Remarks

Этот интерфейс предоставляет методы и свойства, управляющие командами пользовательского интерфейса. `ICommandUI` аналогичен [классу поддержка CCmdUI](../../mfc/reference/ccmdui-class.md), за исключением того, что `ICommandUI` используется для приложений MFC, которые взаимодействуют с компонентами .NET.

`ICommandUI` используется в обработчике ON_UPDATE_COMMAND_UI в классе, производном от [ICommandTarget](../../mfc/reference/icommandtarget-interface.md). Когда пользователь приложения активирует (выбирает или нажимает) меню, каждый пункт меню отображается как включенный или отключенный. Цель каждой команды меню предоставляет эти сведения путем реализации обработчика ON_UPDATE_COMMAND_UI. Для каждого объекта командного интерфейса пользователя в приложении используйте [мастер классов](mfc-class-wizard.md) , чтобы создать запись схемы сообщений и прототип функции для каждого обработчика.

Дополнительные сведения об `ICommandUI` использовании интерфейса в маршрутизации команд см. в разделе [инструкции. Добавление маршрутизации команд в элемент управления Windows Forms](../../dotnet/how-to-add-command-routing-to-the-windows-forms-control.md).

Дополнительные сведения об использовании Windows Forms см. в разделе [Использование пользовательского элемента управления формы Windows в MFC](../../dotnet/using-a-windows-form-user-control-in-mfc.md).

Дополнительные сведения об управлении командами пользовательского интерфейса в MFC см. в разделе [класс поддержка CCmdUI](../../mfc/reference/ccmdui-class.md).

## <a name="icommanduicheck"></a><a name="check"></a> Икоммандуи:: Check

Задает для элемента пользовательского интерфейса для этой команды соответствующее состояние проверки.

```
property UICheckState Check;
```

### <a name="remarks"></a>Remarks

Это свойство задает для элемента пользовательского интерфейса для этой команды соответствующее состояние проверки. Установите флажок проверять следующие значения:

- 0 снять флажок
- 1 Проверка
- 2 задать неопределенное

## <a name="icommanduicontinuerouting"></a><a name="continuerouting"></a> Икоммандуи:: Континуераутинг

Указывает механизму маршрутизации команд продолжить маршрутизацию текущего сообщения вниз по цепочке обработчиков.

```cpp
void ContinueRouting();
```

### <a name="remarks"></a>Remarks

Это расширенная функция-член, которая должна использоваться в сочетании с обработчиком ON_COMMAND_EX, возвращающим значение FALSE. Дополнительные сведения см. в статье Технические примечания TN006: схемы сообщений.

## <a name="icommanduienabled"></a><a name="enabled"></a> Икоммандуи:: Enabled

Включает или отключает элемент пользовательского интерфейса для этой команды.

```
property bool Enabled;
```

### <a name="remarks"></a>Remarks

Это свойство включает или отключает элемент пользовательского интерфейса для этой команды. Установите для параметра Включено значение TRUE, чтобы включить элемент, и FALSE, чтобы отключить его.

## <a name="icommanduiid"></a><a name="id"></a> Икоммандуи:: ID

Возвращает идентификатор объекта пользовательского интерфейса, представленного объектом Икоммандуи.

```
property unsigned int ID;
```

### <a name="remarks"></a>Remarks

Это свойство получает идентификатор (маркер) элемента меню, кнопки панели инструментов или другого объекта пользовательского интерфейса, представленного объектом Икоммандуи.

property unsigned int Index;

### <a name="remarks"></a>Remarks

## <a name="icommanduiradio"></a><a name="radio"></a> Икоммандуи:: Radio

Задает для элемента пользовательского интерфейса для этой команды соответствующее состояние проверки.

```
property bool Radio;
```

### <a name="remarks"></a>Remarks
