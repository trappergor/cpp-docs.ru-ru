# <a name="tree-control-styles"></a>Стили древовидного элемента управления

Стили управления деревом ([CTreeCtrl](../mfc/reference/ctreectrl-class.md)) определяют аспекты внешнего вида элемента управления "дерево". Начальные стили задаются при создании элемента управления "дерево". Стили можно извлечь и изменить после создания элемента управления "дерево" с помощью функций Windows [жетвиндовлонг](/windows/win32/api/winuser/nf-winuser-getwindowlongw) и [SetWindowLong](/windows/win32/api/winuser/nf-winuser-setwindowlongw) , указав **GWL_STYLE** для параметра *ниндекс* . Полный список стилей см. в разделе [стили окна управления древовидным представлением](/windows/win32/Controls/tree-view-control-window-styles) в Windows SDK.

Стиль **TVS_HASLINES** расширяет графическое представление иерархии древовидного элемента управления путем рисования линий, связывающих дочерние элементы с соответствующим родительским элементом. Этот стиль не связывает элементы в корне иерархии. Для этого необходимо объединить стили **TVS_HASLINES** и **TVS_LINESATROOT** .

Пользователь может развернуть или свернуть список дочерних элементов родительского элемента, дважды щелкнув родительский элемент. Элемент управления "дерево", имеющий стиль **TVS_SINGLEEXPAND** , вызывает расширение элемента и невыделенный элемент для свертывания. Если мышь используется для единичного щелчка по выбранному элементу и этот элемент закрыт, он будет раскрыт. Если выбранный элемент является одиночным щелчком мыши, он будет свернут.

Элемент управления "дерево", имеющий стиль **TVS_HASBUTTONS** , добавляет кнопку слева от каждого родительского элемента. Пользователь может нажать кнопку, чтобы развернуть или свернуть дочерние элементы в качестве альтернативы двойному щелчку родительского элемента. **TVS_HASBUTTONS** не добавляет кнопки к элементам в корне иерархии. Для этого необходимо объединить **TVS_HASLINES**, **TVS_LINESATROOT**и **TVS_HASBUTTONS**.

Стиль **TVS_EDITLABELS** позволяет пользователю изменять метки элементов управления "дерево". Дополнительные сведения об изменении меток см. в подразделе [Изменение метки элемента управления дерева](../mfc/tree-control-label-editing.md) далее в этой статье.

Стиль **TVS_NOTOOLTIPS** отключает функцию автоматической подсказки в элементах управления иерархического представления. Эта функция автоматически отображает подсказку, содержащую заголовок элемента под курсором мыши, если все заголовки не видны в данный момент.
