# <a name="making-owner-drawn-header-controls"></a>Создание определяемых владельцем элементов управления "Заголовок"

Можно определить отдельные элементы элемента управления "заголовок" ([CHeaderCtrl](../mfc/reference/cheaderctrl-class.md)), чтобы они были элементами, рисуемыми владельцем. Дополнительные сведения см. в разделе [элементы управления "заголовок, рисуемые владельцем](/windows/win32/Controls/header-controls) " в Windows SDK.
