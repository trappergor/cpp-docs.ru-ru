# <a name="using-cricheditctrl"></a>Использование CRichEditCtrl

Элемент управления форматированным редактированием — это окно, в котором пользователь может вводить и редактировать текст. Текст можно назначить форматированием символов и абзацев и могут включать внедренные объекты OLE. Представления управления форматированным редактированием в MFC, [CRichEditCtrl](../mfc/reference/cricheditctrl-class.md) класса.
