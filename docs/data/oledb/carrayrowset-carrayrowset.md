# <a name="carrayrowsetcarrayrowset"></a>CArrayRowset::CArrayRowset
Создает новый объект `CArrayRowset`.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
      CArrayRowset(   
   int nMax = 100000    
);  
```  
  
#### <a name="parameters"></a>Параметры  
 `nMax`  
 [in] Максимальное число строк в наборе строк.
