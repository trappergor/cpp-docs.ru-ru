# NULL (CRT)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**NULL** \- это пустой указатель, используемый со многими операциями и функциями для указателей.  Он аналогичен 0.  **NULL** определен в следующих файлах заголовка. CRTDBG.H, LOCALE.H, STDDEF.H, STDIO.H, STDLIB.H, STRING.H, TCHAR.H, TIME.H и WCHAR.H.
