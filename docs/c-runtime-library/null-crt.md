# <a name="null-crt"></a>NULL (CRT)

**NULL** представляет собой пустое значение указателя и используется во многих операциях с указателями и функциях. Он эквивалентен 0. Значение **NULL** определено в следующих файлах заголовков: CRTDBG.H, LOCALE.H, STDDEF.H, STDIO.H, STDLIB.H, STRING.H, TCHAR.H, TIME.H и WCHAR.H.
