# <a name="bitand"></a>bitand
Альтернатива оператору &.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
#define bitand &  
  
```  
  
## <a name="remarks"></a>Примечания  
 Макрос создает оператор  
  
## <a name="example"></a>Пример  
  
```  
// iso646_bitand.cpp  
// compile with: /EHsc  
#include <iostream>  
#include <iso646.h>  
  
int main( )  
{  
   using namespace std;  
   int a = 1, b = 2, result;  
  
   result = a & b;  
   cout << result << endl;  
  
   result= a bitand b;  
   cout << result << endl;  
}  
```  
  
```Output  
0  
0  
```
