# <a name="putenv"></a>putenv
Эта функция POSIX устарела. Используйте вместо нее соответствующую стандарту ISO C++ функцию [_putenv](../../c-runtime-library/reference/putenv-wputenv.md) или ее аналог с улучшенной безопасностью [_putenv_s](../../c-runtime-library/reference/putenv-s-wputenv-s.md).  
  
> [!IMPORTANT]
>  Этот API нельзя использовать в приложениях, выполняемых в среде выполнения Windows. Дополнительные сведения см. в разделе [функции CRT, которые не поддерживаются в приложениях универсальной платформы Windows](../../cppcx/crt-functions-not-supported-in-universal-windows-platform-apps.md).
