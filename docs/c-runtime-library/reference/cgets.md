# <a name="cgets"></a>cgets
Эта функция POSIX устарела. Используйте вместо нее соответствующую стандарту ISO C++ функцию [_cgets](../../c-runtime-library/cgets-cgetws.md) или ее аналог с улучшенной безопасностью [_cgets_s](../../c-runtime-library/reference/cgets-s-cgetws-s.md).  
  
> [!IMPORTANT]
>  Этот API нельзя использовать в приложениях, выполняемых в среде выполнения Windows.
