# <a name="ismbbkpunct-ismbbkpunctl"></a>_ismbbkpunct, _ismbbkpunct_l
Проверяет, является ли многобайтовый символ знаком препинания.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
int _ismbbkpunct(  
   unsigned int c   
);  
int _ismbbkpunct_l(  
   unsigned int c,  
   _locale_t locale  
);  
```  
  
#### <a name="parameters"></a>Параметры  
 `c`  
 Целое число, которое требуется проверить.  
  
 `locale`  
 Используемый языковой стандарт.  
  
## <a name="return-value"></a>Возвращаемое значение  
 `_ismbbkpunct` возвращает ненулевое значение, если целое число `c` представляет собой символ знака препинания; в противном случае возвращает 0. Например, только для кодовой страницы 932, `_ismbbkpunct` проверяет на принадлежность к пунктуационным символам катаканы. Функция`_ismbbkpunct` использует текущий языковой стандарт для любых параметров символов, зависящих от языкового стандарта. Функция`_ismbbkpunct_l` идентична, за исключением того, что используется переданный языковой стандарт. Для получения дополнительной информации см. [Locale](../../c-runtime-library/locale.md).
