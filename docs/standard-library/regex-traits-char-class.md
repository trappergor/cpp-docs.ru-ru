# <a name="regex_traitsltchargt-class"></a>Класс regex_traits&lt;char&gt;

Специализация `regex_traits` для **`char`** .

## <a name="syntax"></a>Синтаксис

```cpp
template <>
class regex_traits<char>
```

## <a name="remarks"></a>Remarks

Класс является явной специализацией шаблона класса [regex_traits](../standard-library/regex-traits-class.md) для элементов типа **`char`** (чтобы можно было использовать функции библиотеки, которые управляют объектами этого типа).

## <a name="requirements"></a>Требования

**Заголовок:**\<regex>

**Пространство имен:** std
