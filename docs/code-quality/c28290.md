# <a name="c28290"></a>C28290

> Warning C28290: аннотация функции содержит больше внешних элементов, чем фактическое число параметров

Средство анализа кода сообщает об этом предупреждении, когда аннотация функции содержит больше внешних элементов, чем фактическое число параметров.
