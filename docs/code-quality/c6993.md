# <a name="c6993"></a>C6993

> предупреждение C6993: анализ кода не учитывает конструкции OpenMP; анализ кода с одним потоком

Это предупреждение означает, что анализатор кода обнаружил открытые директивы pragma MP, которые не удалось проанализировать.
