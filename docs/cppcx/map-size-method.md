# Метод Map::Size
Возвращает количество элементов [Windows::Foundation::Collections::IKeyValuePair\<K,V\>](http://msdn.microsoft.com/library/windows/apps/br226031.aspx) в объекте Map.  
  
## Синтаксис  
  
```  
  
virtual property unsigned int Size;  
```  
  
## Возвращаемое значение  
 Количество элементов в объекте Map.  
  
## Требования  
 **Заголовок:** collection.h  
  
 **Пространство имен:** Platform::Collections
