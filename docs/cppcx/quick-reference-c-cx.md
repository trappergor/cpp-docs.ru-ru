# <a name="quick-reference-ccx"></a>Краткий справочник (C++/CX)
Среда выполнения Windows поддерживает приложений универсальной платформы Windows, которые выполняются только в доверенной среде операционной системы, используют авторизованные функции, типы данных и устройства и распространяются через [!INCLUDE[win8_appstore_long](../cppcx/includes/win8-appstore-long-md.md)]. C + +/ CX упрощают создание приложений для среды выполнения Windows. В этой статье приводится краткий справочник; более полную документацию см. в разделе [системы типов](../cppcx/type-system-c-cx.md) и [расширения компонентов для платформ среды выполнения](http://go.microsoft.com/fwlink/p/?linkid=228720).  
  
 При построении в командной строке используйте **/zw** параметр компилятора для построения приложения универсальной платформы Windows или компонента среды выполнения Windows. Для доступа к объявления среды выполнения Windows, которые определены в файлах метаданных (.winmd) для среды выполнения Windows, укажите `#using` директива или **/FU** параметр компилятора. При создании проекта для приложения универсальной платформы Windows по умолчанию Visual Studio устанавливает эти параметры и добавляет ссылки на все библиотеки среды выполнения Windows.
