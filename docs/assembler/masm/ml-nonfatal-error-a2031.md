# ML Nonfatal Error A2031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**следует индекс или базовый регистр**  
  
 Предпринята попытка использовать регистр, который не был регистром базовому или индекса в выражении памяти.  
  
 Например, следующие выражения вызывают эту ошибку.  
  
```  
[ax]  
[bl]  
```
