# TEXTEQU
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Присвоити *textitem* В *Имя*.   Textitem может быть символьная строка, константа предшествуемая a `%`или строка, возвращаемых функцией макроса.  
  
## Синтаксис  
  
```  
  
name TEXTEQU [[textitem]]  
```
