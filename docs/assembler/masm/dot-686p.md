# <a name="686p-32-bit-masm"></a>.686P (32-разрядный компилятор MASM)

Включает сборку всех инструкций (включая привилегированные) для процессора Pentium Pro. (только 32-разрядный MASM.)

## <a name="syntax"></a>Синтаксис

> **.686P**
