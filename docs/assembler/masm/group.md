# GROUP
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Добавьте определенное *сегменты* к вызываемой группе *Имя*.  
  
## Синтаксис  
  
```  
  
name GROUP segment [[, segment]]...  
```
