# <a name="ml-nonfatal-error-a2044"></a>Некритичная ошибка ML A2044
**Недопустимый символ в файл**  
  
 Исходный файл содержит символ за пределами комментария, строки, или литерала, не распознан как оператор или другие юридические символов.
