# ML Fatal Error A1009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**линия слишком много времени**  
  
 Линия в исходном файле превышает предел \- 512 символов.  
  
 Если сцеплены несколько физических линии со знака линия\-продолжения \(\\\), то результат логической линия по\-прежнему ограничена до 512 символов.
