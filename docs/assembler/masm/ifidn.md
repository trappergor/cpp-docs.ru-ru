# IFIDN]
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Предоставляет сборку, если элементы текста совпадают.  
  
## Синтаксис  
  
```  
  
IFIDN[[I]] textitem1, textitem2  
```  
  
## Заметки  
 If **е** обеспечивает сравнение нечувствительно к регистру.  См. IF для полного синтаксиса.
