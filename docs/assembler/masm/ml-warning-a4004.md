# <a name="ml-warning-a4004"></a>Предупреждение ML A4004

**невозможно ПРЕДПОЛОЖИТЬ CS**

Предпринята попытка предположить значение для регистра CS. CS всегда устанавливается в текущий сегмент или группу.
