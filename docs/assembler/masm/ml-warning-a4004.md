# <a name="ml-warning-a4004"></a>Предупреждение ML A4004
**не удается ПРЕДПОЛАГАЕТСЯ CS**  
  
 Была предпринята попытка предположить значение регистра CS. CS всегда равно текущему сегменту или группы.
