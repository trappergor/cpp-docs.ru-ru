# ML Warning A4004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**не удалось ВЫПОЛНИТЬ CS**  
  
 Предпринята попытка выполнить значение регистра CS.  CS всегда устанавливается в текущую сегмент или группе.
