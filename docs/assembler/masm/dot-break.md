# <a name="break"></a>.BREAK
Создает код для завершения [. ХОТЯ](../../assembler/masm/dot-while.md) или [. ПОВТОРИТЕ](../../assembler/masm/dot-repeat.md) блокировать, если *условие* имеет значение true.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
.BREAK [[.IF condition]]   
```
