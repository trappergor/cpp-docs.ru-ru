# <a name="qword"></a>QWORD
Выделяет и при необходимости инициализирует 8 байтов памяти для каждого `initializer`. Также можно использовать в качестве спецификатора типа в любом является допустимым типом.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
[[name]] QWORD initializer [[, initializer]]...  
```
