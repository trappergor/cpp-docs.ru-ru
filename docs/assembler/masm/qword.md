# QWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Выделяет и инициализирует 8 байт хранения для каждого `initializer`.  Также может использоваться как описатель типа везде, где допустим тип.  
  
## Синтаксис  
  
```  
[[name]] QWORD initializer [[, initializer]]...  
```
