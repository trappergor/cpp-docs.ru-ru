# .LISTMACROALL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Запускает список всех выписок в макросах.  
  
## Синтаксис  
  
```  
  
.LISTMACROALL  
  
```  
  
## Заметки  
 Аналогично .LALL.
