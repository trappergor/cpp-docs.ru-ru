# <a name="ml-nonfatal-error-a2047"></a>Некритичная ошибка ML A2047

**пустая строка (null)**

Строка, состоящая из пары знаков кавычек и без символов в.

Чтобы строка была допустимой, она должна содержать 1-255 символов.
