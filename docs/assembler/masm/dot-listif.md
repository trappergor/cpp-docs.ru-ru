# <a name="listif"></a>.LISTIF

Запускает список инструкций в ложных условных блоках.

## <a name="syntax"></a>Синтаксис

> **.LISTIF**

## <a name="remarks"></a>Примечания

То же, что и [. ЛФКОНД](../../assembler/masm/dot-lfcond.md).
