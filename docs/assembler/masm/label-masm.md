# <a name="label-masm"></a>LABEL (MASM)
Создает новую метку, назначив текущее значение счетчика расположение и заданного `type` для *имя*.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
      name LABEL type  
name LABEL [[NEAR | FAR | PROC]] PTR [[type]]   
```
