# <a name="repeat"></a>REPEAT
Помечает блок, который должен повторяться *выражение* раз. То же, что [REPT](../../assembler/masm/rept.md).  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
   REPEAT expression  
statements  
ENDM  
```
