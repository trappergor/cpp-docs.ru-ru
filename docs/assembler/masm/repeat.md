# <a name="repeat"></a>REPEAT

Помечает блок, который должен повторяться *выражение* раз. Совпадение с кодом [REPT](../../assembler/masm/rept.md).

## <a name="syntax"></a>Синтаксис

> ПОВТОРИТЕ *выражение*<br/>
> *Инструкции*<br/>
> ENDM
