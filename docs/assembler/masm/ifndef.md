# IFNDEF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Предоставляет сборку, если *Имя* не указано.  
  
## Синтаксис  
  
```  
  
IFNDEF name  
```  
  
## Заметки  
 См. IF для полного синтаксиса.
