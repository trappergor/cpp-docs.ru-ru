# <a name="ifndef"></a>IFNDEF

Предоставляет сборку, если *имя* не был определен.

## <a name="syntax"></a>Синтаксис

> IFNDEF имя

## <a name="remarks"></a>Примечания

См. в разделе [IF](../../assembler/masm/if-masm.md) сложный синтаксис.
