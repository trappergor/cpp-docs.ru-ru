# <a name="operator-width"></a>Оператор WIDTH

Возвращает ширину в битах текущего *рекордфиелднаме* или *записи*.

## <a name="syntax"></a>Синтаксис

> **Ширина** {*рекордфиелднаме* | *запись*}
