# <a name="operator-mask"></a>Оператор MASK

Returns a bit mask in which the bits in *recordfieldname* or *record* are set and all other bits are cleared.

## <a name="syntax"></a>Синтаксис

> **MASK** {*recordfieldname* | *record*}
