# <a name="ml-nonfatal-error-a2059"></a>Некритичная ошибка ML A2059
**принудительно ошибка: равенства строк**  
  
 Ошибка условной директивы [. ERRIDN](../../assembler/masm/dot-erridn.md) или **. ERRIDNI** использовался для создания этой ошибки.
