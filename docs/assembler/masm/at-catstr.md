# <a name="catstr"></a>@CatStr
Макрос функция, которая объединяет один или несколько строк. Возвращает строку.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
@CatStr( string1 [[, string2...]] )  
```
