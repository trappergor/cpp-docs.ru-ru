# WHILE (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Повторяющиеся сборка блока `statements` пока Выражение остается истинными.  
  
## Синтаксис  
  
```  
  
   WHILE expression  
statements  
ENDM  
```
