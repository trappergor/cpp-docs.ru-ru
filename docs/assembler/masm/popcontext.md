# <a name="popcontext"></a>POPCONTEXT
Восстановление всех или части текущего `context` (автор [PUSHCONTEXT](../../assembler/masm/pushcontext.md) директива). `context` Может быть **ПРЕДПОЛАГАЕТСЯ, что**, `RADIX`, **ВЫВОД**, **ЦП**, или **все**.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
POPCONTEXT context  
```
