# <a name="operator-"></a>operator +
Первый оператор возвращает *expression1* , а также *expression2*. Второй оператор обращает знак *выражение*.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
      expression1 + expression2  
-expression  
```
