# <a name="exit"></a>.EXIT
Создает код завершения.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
.EXIT [[expression]]  
```  
  
## <a name="remarks"></a>Примечания  
 Возвращает необязательный *выражение* оболочку.
