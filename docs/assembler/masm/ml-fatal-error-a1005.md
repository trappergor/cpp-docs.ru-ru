# ML Fatal Error A1005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**ограничение ассемблера: таблица полного имени параметра макроса**  
  
 Слишком много параметров, локальные или меток макроса для макроса.  Была нет больше места в таблице имени макроса.  
  
 Укажите мало или меньше имя или удалите ненужные макросов.
