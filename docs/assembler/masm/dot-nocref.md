# .NOCREF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Подавляет список символов в файле таблицы и браузера символов.  
  
## Синтаксис  
  
```  
  
.NOCREF [[name[[, name]]...]]  
```  
  
## Заметки  
 Если имена определяются только заданного имена подавляются.  Аналогично .XCREF.
