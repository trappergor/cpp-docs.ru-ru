# <a name="operator-xor"></a>Оператор XOR
Возвращает результат побитовой операции **XOR** операции для *expression1* и *expression2*.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
expression1 XOR expression2  
```
