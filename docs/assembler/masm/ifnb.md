# IFNB
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Предоставляет сборку, если *textitem* не пустыми.  
  
## Синтаксис  
  
```  
  
IFNB textitem  
```  
  
## Заметки  
 См. IF для полного синтаксиса.
