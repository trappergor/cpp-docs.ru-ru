# <a name="operator-lowword"></a>Оператор LOWWORD
Возвращает младшие 16 разрядов *выражение*. Выражения MASM являются 64-разрядными значениями.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
LOWWORD expression  
```
