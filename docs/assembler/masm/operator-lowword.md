# <a name="operator-lowword"></a>Оператор LOWWORD

Возвращает младшие 16 разрядов *выражения*. Выражения MASM — это 64-разрядные значения.

## <a name="syntax"></a>Синтаксис

> *Выражение* LOWWORD
