# ML Nonfatal Error A2189
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**недопустимое сочетание с выравниванием сегмента**  
  
 Выравнивание указанный **Выровнять** OR  `EVEN` директива была больше, чем текущее выравнивание сегмента определяется как  **Сегмент** директива.
