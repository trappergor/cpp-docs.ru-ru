# <a name="nolistmacro"></a>.NOLISTMACRO

Подавляет список расширения макроса.

## <a name="syntax"></a>Синтаксис

> .NOLISTMACRO

## <a name="remarks"></a>Примечания

Совпадение с кодом [. SALL](../../assembler/masm/dot-sall.md).
