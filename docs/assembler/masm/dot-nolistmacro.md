# <a name="nolistmacro"></a>.NOLISTMACRO
Подавляет перечень расширения макросов.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
.NOLISTMACRO  
  
```  
  
## <a name="remarks"></a>Примечания  
 То же, что [. SALL](../../assembler/masm/dot-sall.md).
