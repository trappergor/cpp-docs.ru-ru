# <a name="ml-nonfatal-error-a2206"></a>Некритичная ошибка ML A2206
**отсутствует оператор в выражении**  
  
 Невозможно вычислить выражение, поскольку отсутствует оператор. Это сообщение об ошибке также может быть следствием предыдущей ошибки в программе.  
  
 Следующая строка приведет к возникновению этой ошибки:  
  
```  
value1 = ( 1 + 2 ) 3  
  
```
