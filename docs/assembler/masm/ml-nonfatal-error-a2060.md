# <a name="ml-nonfatal-error-a2060"></a>Некритичная ошибка ML A2060
**принудительно ошибка: строки не равны.**  
  
 Ошибка условной директивы [. ERRDIF](../../assembler/masm/dot-errdif.md) или **. ERRDIFI** использовался для создания этой ошибки.
