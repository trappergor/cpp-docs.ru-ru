# <a name="endp"></a>ENDP

Помечает конец процедуры *имя* ранее начавшиеся с **PROC**.

## <a name="syntax"></a>Синтаксис

> *имя* ENDP

## <a name="remarks"></a>Примечания

См. в разделе [PROC](../../assembler/masm/proc.md).
