# ML Nonfatal Error A2004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**конфликт типа символов: Идентификатор**  
  
 [EXTERNDEF](../../assembler/masm/externdef.md) OR  [МЕТКА](../../assembler/masm/label-masm.md) директива использовалась на символе, переменной, структуре данных или метке, которая была определена в одном модуле, но разным типом.
