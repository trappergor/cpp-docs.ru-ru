# ML Nonfatal Error A2066
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**несовместимые режим ЦП и размер сегмента**  
  
 Предпринята попытка открыть сегмент with a **USE16**"  **USE32**или  **Плоский** атрибут, который не был совместим с указанным ЦП или изменять в 16\-разрядное ЦП в сегменте во время обновления 32 \(sp2\).  
  
 **USE32** и  **Плоский** атрибуты должны стоять .386 или больше директивы процессора.
