# <a name="linker-tools-warning-lnk4065"></a>Предупреждение средств компоновщика LNK4065

"функция" не может быть упорядочена; игнорируют

Данная функция не была скомпилирована как упакованная функция. Перекомпилируйте с использованием [/Gy](../../build/reference/gy-enable-function-level-linking.md).
