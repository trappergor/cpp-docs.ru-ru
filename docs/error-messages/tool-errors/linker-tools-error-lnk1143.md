# <a name="linker-tools-error-lnk1143"></a>Ошибка средств компоновщика LNK1143
Недопустимый или поврежденный файл: отсутствует символ для секции COMDAT  
  
 Эта ошибка может возникнуть, если объектный файл поврежден.  
  
 Перестройте файл.
