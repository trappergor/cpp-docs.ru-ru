# <a name="linker-tools-error-lnk1143"></a>Ошибка средств компоновщика LNK1143

Недопустимый или поврежденный файл: нет символа для секции COMDAT

Эта ошибка может быть вызвана, если объектный файл поврежден.

Перестройте файл.
