# <a name="linker-tools-error-lnk1113"></a>Ошибка средств компоновщика LNK1113

Недопустимый тип тип компьютера

Тип компьютера, указанный в заголовке объекта не является допустимым.

Эта ошибка может быть вызвана, если файл поврежден. Построение объекта.
