# <a name="resource-compiler-fatal-error-rc1004"></a>Неустранимая ошибка компилятора ресурсов RC1004

обнаружен непредвиденный конец файла

Эта ошибка может быть вызвана отсутствием перевода строки и символами возврата каретки в последней строке текстового файла.
