# Ошибка вычислителя выражений CXX0051
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

в точке останова указано несколько перегруженных символов  
  
 Вычислителю выражений не удалось определить, какой перегруженный символ из указанных следует использовать в качестве точки останова.  
  
 Эта ошибка идентична CAN0051.
