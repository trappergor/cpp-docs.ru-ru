# <a name="expression-evaluator-error-cxx0031"></a>Ошибка вычислителя выражений CXX0031

выражение не подлежит расширению

В процессе вычисления выражения произошла внутренняя ошибка.

Можно написать эквивалентное выражение, которое может быть вычислено правильно.

Эта ошибка идентична ошибке CAN0031.
