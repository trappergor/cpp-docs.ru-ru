# <a name="resource-compiler-fatal-error-rc1009"></a>Неустранимая ошибка компилятора ресурсов RC1009

ограничение компилятора: недопустимая степень вложения «макрос»

Файл превышает предел компилятора ресурсов для вложения макросов. Проверьте указанный исходный файл для уменьшения глубины вложения макросов.
