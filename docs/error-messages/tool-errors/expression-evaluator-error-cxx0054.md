# <a name="expression-evaluator-error-cxx0054"></a>Ошибка вычислителя выражений CXX0054

Недопустимое регистрация в выражении точки останова

Регистр нельзя использовать в выражении точки останова.

Эта ошибка идентична CAN0054.
