# Неустранимая ошибка компилятора ресурсов RC1205
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

недопустимая кодовая страница  
  
 После параметра "Определение кодовой страницы" \(\/c\) следует недопустимая кодовая страница.  Дополнительные сведения см. в разделе [Кодовые страницы](../../c-runtime-library/code-pages.md) раздела *Справка по библиотеке времени выполнения*.
