# <a name="profile-guided-optimization-warning-pg1087"></a>Предупреждение профильной оптимизации PG1087
PGC-файл «файл» не является допустимым.  
  
 PGC-файл поврежден, возможно, вследствие аварийного завершения сеанса профилирования.  Повторно запустите сценарий профилирования и завершается.
