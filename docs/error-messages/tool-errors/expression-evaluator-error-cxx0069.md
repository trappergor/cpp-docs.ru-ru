# <a name="expression-evaluator-error-cxx0069"></a>Ошибка вычислителя выражений CXX0069

для переменной требуется кадр стека

Вычислитель выражений не может вычислить переменную, так как она не встречается в кадре стека. Это может быть вызвано переменными, объявленными как часть встроенной функции.
