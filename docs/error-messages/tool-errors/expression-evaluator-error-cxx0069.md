# Ошибка вычислителя выражений CXX0069
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

для переменной требуется кадр стека  
  
 Вычислитель выражений не смог вычислить переменную, поскольку она отсутствует во кадре стека.  Это может происходить при объявлении переменных внутри встраиваемых функций.
