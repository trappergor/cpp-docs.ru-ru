# Ошибка вычислителя выражений CXX0014
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

отсутствующий операнд  
  
 В операторе не указан обязательный операнд.  Проверьте синтаксис выражения.  
  
 Эта ошибка идентична CAN0014.
