# <a name="linker-tools-error-lnk2026"></a>Ошибка средств компоновщика LNK2026
небезопасный модуль для образа SAFESEH  
  
 [/ Параметр SAFESEH](../../build/reference/safeseh-image-has-safe-exception-handlers.md) был указан, но модуль несовместим с функцией безопасной обработки исключений. Если вы хотите использовать этот модуль с параметром **/SAFESEH**, необходимо выполнить повторную компиляцию модуля.
