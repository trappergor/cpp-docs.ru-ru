# <a name="linker-tools-warning-lnk4199"></a>Предупреждение средств компоновщика LNK4199

ПАРАМЕТР/DELAYLOAD: dllname игнорируется; не найдены импорты из dllname

Компоновщик игнорирует `dllname`, поскольку для него не требуются функции, `dllname` экспорты.
