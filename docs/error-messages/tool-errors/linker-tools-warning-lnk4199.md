# <a name="linker-tools-warning-lnk4199"></a>Предупреждение средств компоновщика LNK4199

/DELAYLOAD:dllname игнорируется; не найдены операции импорта из dllname

Компоновщик не обрабатывает `dllname` так, как любую из функций, не требуется, `dllname` экспортирует.
