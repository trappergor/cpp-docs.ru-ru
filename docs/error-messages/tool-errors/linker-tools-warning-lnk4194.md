# <a name="linker-tools-warning-lnk4194"></a>Предупреждение средств компоновщика LNK4194

Имя /DELAYLOAD:DLL игнорируется

Компоновщик не поддерживает [отложенную загрузку](../../build/reference/delayload-delay-load-import.md) запрошенной библиотеки DLL.
