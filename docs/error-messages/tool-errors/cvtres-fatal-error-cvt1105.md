# <a name="cvtres-fatal-error-cvt1105"></a>Неустранимая ошибка CVTRES CVT1105

не удается выполнить поиск в файле

CVTRES не удается перейти в расположение в файле.

Эта ошибка может быть вызвана повреждения файлов.
