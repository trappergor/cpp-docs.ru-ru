# Неустранимая ошибка CVTRES CVT1105
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

не удается выполнить поиск в файле  
  
 CVTRES не удается перейти к заданному расположению в файле.  
  
 Данная ошибка может быть вызвана повреждением файла.
