# <a name="nmake-fatal-error-u1034"></a>Неустранимая ошибка NMAKE U1034
Синтаксическая ошибка: отсутствие разделителя  
  
 Двоеточие (**:**), отделяет целевые объекты и зависимые объекты отсутствует.
