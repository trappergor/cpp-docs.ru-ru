# <a name="nmake-fatal-error-u1086"></a>Неустранимая ошибка NMAKE U1086
**правило вывода не может иметь зависимости**  
  
 Двоеточие (**:**) в определение правила следует указать один из следующих:  
  
-   Символ перевода строки  
  
-   Точка с запятой (**;**), который можно выполнять с помощью команды  
  
-   Знак номера (**#**), который может следовать комментарий
