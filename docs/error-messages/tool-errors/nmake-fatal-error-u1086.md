# <a name="nmake-fatal-error-u1086"></a>Неустранимая ошибка NMAKE U1086
**правило определения не может иметь зависимости**  
  
 Двоеточие (**:**) в определение правила должен следовать один из следующих:  
  
-   Символ новой строки  
  
-   Точка с запятой (**;**), которой может следовать команда  
  
-   Знак номера (**#**), которой может следовать комментарий
