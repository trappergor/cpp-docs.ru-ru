# <a name="nmake-fatal-error-u1083"></a>Неустранимая ошибка NMAKE U1083

подстановки конечный макрос «target»

Данный целевой объект является вызовом макроса, который не был определен или имеет значение null. NMAKE не удается обработать целевой объект null.
