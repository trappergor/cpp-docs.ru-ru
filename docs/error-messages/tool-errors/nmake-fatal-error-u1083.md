# <a name="nmake-fatal-error-u1083"></a>Неустранимая ошибка NMAKE U1083
макрос 'target' разворачивается в значение nothing  
  
 Данный целевой объект является вызовом макроса, который не определен или имеет значение null. NMAKE не удается обработать целевой объект null.
