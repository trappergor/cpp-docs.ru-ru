# Ошибка вычислителя выражений CXX0047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Список аргументов не соответствует функции  
  
 Выражение вызвало функцию с фактическим списком параметров, не соответствующим формальному списку параметров какой\-либо функции с таким же именем, определенной в программе.  
  
 Перегруженные функции могут быть вызваны только в случае точного соответствия параметров или при условии, что соответствие не требует построения объекта.  
  
 Эта ошибка идентична ошибке CAN0047.
