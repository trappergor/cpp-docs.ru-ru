# <a name="resource-compiler-error-rc2171"></a>Ошибка компилятора ресурсов RC2171

Неизвестный формат DIB заголовка

Заголовок растровое изображение не **BITMAPCOREHEADER** или **BITMAPINFOHEADER** структуры.
