# <a name="expression-evaluator-error-cxx0034"></a>Ошибка вычислителя выражений CXX0034

**типы несовместимы с оператором**

Указанные типы операндов не допустимы для операции.

Например указатель нельзя будет умножено на любое значение.

Возможно, вам нужно ввести операнды к типу, совместимому с помощью оператора приведения.
