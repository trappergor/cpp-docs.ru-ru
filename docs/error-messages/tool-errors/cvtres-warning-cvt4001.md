# <a name="cvtres-warning-cvt4001"></a>Предупреждение CVTRES CVT4001
**Тип компьютера, не указан; предполагается, что**   
 ***type***  
  
 CVTRES не нашла спецификацию машины. Предполагается, что тип данного компьютера. Если значение по умолчанию — неверен, перезапустите CVTRES с параметром/Machine.
