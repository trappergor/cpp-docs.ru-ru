# <a name="linker-tools-error-lnk1240"></a>Ошибка средств компоновщика LNK1240

не удалось скомпилировать содержимое IDL

Компоновщик породил MIDL для компиляции внедренного IDL, но возникла проблема. Проверьте ошибки, заданные MIDL.
