# <a name="linker-tools-error-lnk1240"></a>Ошибка средств компоновщика LNK1240
не удалось откомпилировать IDL-содержимое  
  
 Компоновщик порожденных MIDL для компиляции внедренный IDL, но возникла проблема. Проверьте наличие ошибок компилятора MIDL.
