# <a name="linker-tools-warning-lnk4200"></a>Предупреждение средств компоновщика LNK4200
поврежденная информация номера строки в объектном файле; обрабатывается  
  
 Сведения о номерах строк в объектном файле повреждена. Выполните перестроение.
