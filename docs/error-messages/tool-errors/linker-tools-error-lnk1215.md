# Ошибка средств компоновщика LNK1215
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

сбой операции метаданных \(HRESULT\) : ошибка  
  
 Компоновщик получил ошибку от среды выполнения .NET при попытке выполнения обновления метаданных через среду выполнения .NET.  
  
 `HRESULT` — это параметр HRESULT из метода среды выполнения .NET.  `error` — это текст, предоставленный .NET.  
  
 Возможно, компоновщик не соответствует среде выполнения .NET; переустановите Visual C\+\+.
