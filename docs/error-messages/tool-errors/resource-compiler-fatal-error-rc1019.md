# <a name="resource-compiler-fatal-error-rc1019"></a>Неустранимая ошибка компилятора ресурсов RC1019

непредвиденный "#else"

Директива `#else` не появилась в конструкции `#if`, **#ifdef**или **#ifndef** .

Убедитесь, что перед этим оператором действует `#if`, **#ifdef**или **#ifndef** .
