# Ошибка компилятора ресурсов RC2165
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

строка не найдена в инструкции DLGINCLUDE  
  
 Инструкция указывает неправильный включенный файл.  
  
 Инструкция **DLGINCLUDE** инструкция должна использовать следующий синтаксис:  
  
 **DLGINCLUDE "*filename*.h"**.
