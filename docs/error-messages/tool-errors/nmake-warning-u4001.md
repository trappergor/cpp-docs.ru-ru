# <a name="nmake-warning-u4001"></a>Предупреждение программы NMAKE U4001

> командный файл может быть вызван только из командной строки

Командный файл, который вызывается спецификатором at ( **\@** ), не может содержать спецификацию для другого командного файла. Такое вложение запрещено. Спецификация пропущена.
