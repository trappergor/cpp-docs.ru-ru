# <a name="resource-compiler-fatal-error-rc1047"></a>Неустранимая ошибка компилятора ресурсов RC1047
«слишком много параметр параметры «строка»»  
  
 Данный параметр был указан слишком много раз. Данная строка — это аргумент параметра, вызвавшего ошибку.
