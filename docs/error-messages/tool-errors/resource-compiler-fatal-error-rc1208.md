# Неустранимая ошибка компилятора ресурсов RC1208
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

входной файл имеет расширение RES  
  
 Расширение RES используется для выходных файлов компилятора ресурсов.  Для скриптов компилятора ресурсов следует использовать расширение RC.
