# <a name="resource-compiler-fatal-error-rc1208"></a>Неустранимая ошибка компилятора ресурсов RC1208

входной файл имеет. Расширение RES

Тот. Для вывода компилятора ресурсов используется расширение RES. Тот. Для скриптов компилятора ресурсов следует использовать расширение RC.
