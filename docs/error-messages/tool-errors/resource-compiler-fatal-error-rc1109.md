# <a name="resource-compiler-fatal-error-rc1109"></a>Неустранимая ошибка компилятора ресурсов RC1109

Ошибка при создании имени ресурса

Не удалось создать указанный RES — файл. Убедитесь, что он не создается на диске только для чтения. Используйте параметр/V, чтобы узнать, создается ли файл.
