# <a name="resource-compiler-fatal-error-rc1018"></a>Неустранимая ошибка компилятора ресурсов RC1018

непредвиденный "#elif"

Директива `#elif` не появилась в конструкции `#if`, **#ifdef**или **#ifndef** .

Убедитесь, что перед этим оператором действует `#if`, **#ifdef**или **#ifndef** .
