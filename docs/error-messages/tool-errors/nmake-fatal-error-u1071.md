# <a name="nmake-fatal-error-u1071"></a>Неустранимая ошибка NMAKE U1071
цикл в дереве зависимостей для целевого объекта «targetname»  
  
 В дереве зависимостей для заданного целевого объекта существует циклическая зависимость. Данный целевой объект является зависит от одного из зависимых элементов для заданного целевого объекта. Циклические зависимости являются недопустимыми.
