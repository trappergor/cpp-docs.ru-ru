# <a name="bscmake-error-bk1510"></a>Ошибка BSCMAKE BK1510
поврежден. Имя SBR-файла  
  
 Данный SBR-файл имеет неправильный формат. Выполните повторную компиляцию, чтобы создать новый SBR-файл.
