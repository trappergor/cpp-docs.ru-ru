# <a name="resource-compiler-error-rc2167"></a>Ошибка компилятора ресурсов RC2167

Нераспознанное поле VERSIONINFO; Ожидается BEGIN или запятая

В **фиксированной** части объявления структуры **versionInfo** обнаружено Нераспознанное поле. Поле **versionInfo** должно быть заключено в DWORD, разделенное запятыми.
