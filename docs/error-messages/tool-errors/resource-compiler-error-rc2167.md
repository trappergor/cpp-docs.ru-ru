# <a name="resource-compiler-error-rc2167"></a>Ошибка компилятора ресурсов RC2167
Неизвестное поле VERSIONINFO; Ожидается BEGIN или запятая  
  
 Неизвестное поле был найден в **FIXED** частью **VERSIONINFO** объявление структуры. Объект **VERSIONINFO** поле должно быть слова DWORD, разделенные запятой.
