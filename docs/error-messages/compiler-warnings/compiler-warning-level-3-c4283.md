# <a name="compiler-warning-level-3-c4283"></a>Предупреждение компилятора (уровень 3) C4283
и через тип «тип»  
  
 Продолжение предупреждения [C4281](../../error-messages/compiler-warnings/compiler-warning-level-3-c4281.md) показывает, что **оператор->** вызывает сам себя через `type`.
