# <a name="compiler-warning-level-1-c4048"></a>Предупреждение (уровень 1) C4048 компилятора
другой объявлен индексы массива: «идентификатор1» и «идентификатор2»  
  
 Выражения используются указатели на массивы разного размера. Указатели используются без преобразования.  
  
 Чтобы устранить это предупреждение, явное приведение массивов в ту же или эквивалентную тип.
