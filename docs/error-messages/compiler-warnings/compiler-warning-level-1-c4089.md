# <a name="compiler-warning-level-1-c4089"></a>Предупреждение (уровень 1) C4089 компилятора
«функция»: различные типы в фактическом параметре «номер» формальный параметр «число»  
  
 Соответствующие формальные и фактические параметры имеют разные типы. Фактический параметр передается без изменений. Функция приводит фактический параметр типа, указанного в определении функции.
