# Предупреждение компилятора (уровень&#160;1) C4164
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"идентификатор": встроенная функция не объявлена  
  
 Указанная встроенная функция не объявлена; может потребоваться включить соответствующий файл заголовка с помощью директивы \#include.
