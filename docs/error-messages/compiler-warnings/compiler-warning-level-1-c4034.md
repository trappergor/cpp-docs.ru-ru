# <a name="compiler-warning-level-1-c4034"></a>Компилятор C4034 предупреждение (уровень 1)
Возвращает значение 0, sizeof  
  
 `sizeof` Оператор применяется к операнду нулевого размера (пустой структуры, объединения, класса или перечисляемый тип или тип `void`).
