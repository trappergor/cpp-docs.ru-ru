# <a name="compiler-warning-level-1-c4034"></a>Предупреждение компилятора (уровень 1) C4034

sizeof возвращает 0

**`sizeof`** Оператор применяется к операнду нулевого размера (пустой структуре, объединению, классу, типу перечисления или типу **`void`** ).
