# Предупреждение компилятора (уровень 4) C4296
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"оператор" : выражение всегда имеет ложное значение  
  
 В операции сравнения с нулем использовалась беззнаковая переменная.  
  
 Данное предупреждение по умолчанию отключено.  Дополнительные сведения см. в разделе [Отключенные по умолчанию предупреждения компилятора](../Topic/Compiler%20Warnings%20That%20Are%20Off%20by%20Default.md).  
  
 Следующий пример приводит к возникновению ошибки C4296:  
  
```  
// C4296.cpp  
// compile with: /W4  
#pragma warning(default : 4296)  
int main()  
{  
   unsigned int u = 9;  
   if (u < 0)    // C4296  
      u++;  
   if (u >= 0)   // C4296  
      u++;  
}  
```
