# <a name="compiler-warning-level-1-c4088"></a>Компилятор C4088 предупреждение (уровень 1)
«функция»: несоответствие указателей в фактическом параметре «номер» формальный параметр «число»  
  
 Соответствующие формальные и фактические параметры имеют разные уровни косвенного обращения. Фактический параметр передается без изменений. Вызываемая функция воспринимает его значение как указатель.
