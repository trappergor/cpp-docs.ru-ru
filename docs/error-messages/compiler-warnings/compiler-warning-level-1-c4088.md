# <a name="compiler-warning-level-1-c4088"></a>Компилятор предупреждение (уровень 1) C4088

«функция»: несоответствие указателей в фактическом параметре «номер», формальный параметр «число»

Соответствующих формальных и фактических параметров имеют разные уровни косвенного обращения. Фактический параметр передается без изменений. Вызываемая функция воспринимает его значение как указатель.
