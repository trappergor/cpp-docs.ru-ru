# Предупреждение компилятора (уровень 1) C4410
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

недопустимый размер для операнда  
  
 Недопустимый размер одного из операторов инструкции.  В этом случае используется наименьший допустимый размер операнда.
