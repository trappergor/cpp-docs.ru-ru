# Предупреждение компилятора (уровень 1) C4024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"функция": различные типы для формального и фактического параметров "номер"  
  
 Соответствующие формальные и фактические параметры имеют различные типы.  Компилятор передает фактический параметр без изменений.  Принимающая функция преобразует тип параметра к ожидаемому типу.
