# <a name="compiler-warning-level-1-c4029"></a>Предупреждение (уровень 1) C4029 компилятора
объявленный список формальных параметров отличается от объявления  
  
 Типы формальных параметров в объявлении функции не соответствуют типам в определении функции. Компилятор использует список параметров из определения.
