# <a name="compiler-warning-level-1-c4122"></a>Предупреждение компилятора (уровень 1) C4122

"функция": alloc_text применимо только к функциям с С-компоновкой

Директива **alloc_text** применяется только к функциям, объявленным с модификатором **extern c**. Ее нельзя использовать с внешними функциями C++.

Прагма игнорируется.
