# Предупреждение компилятора (уровень 1) C4031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

второй список формальных параметров длиннее первого  
  
 Повторное объявление функции с использованием других формальных параметров.  Компилятор использует формальные параметры, заданные в первом объявлении.
