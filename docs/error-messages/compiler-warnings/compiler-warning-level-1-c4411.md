# Предупреждение компилятора (уровень 1) C4411
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"идентификатор": символ соответствует регистру смещения  
  
 Идентификатор является локальным символом, который разрешается в регистр замещения, и поэтому может использоваться в операнде с другим символом.
