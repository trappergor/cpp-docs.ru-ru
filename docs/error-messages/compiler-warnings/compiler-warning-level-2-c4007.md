# <a name="compiler-warning-level-2-c4007"></a>Предупреждение компилятора (уровень 2) C4007

"функция": должен быть "атрибут"

Обязательный атрибут для функции не указан явно. Например, функция **Main** должна иметь атрибут `__cdecl`. Компилятор вызывает принудительное выполнение атрибута.
