# <a name="compiler-warning-level-1-c4402"></a>Предупреждение компилятора (уровень 1) C4402
необходимо использовать оператор PTR  
  
 Тип применяется к операнду без оператора PTR при обращении или преобразовании в тип из встроенного кода ассемблера.  
  
 Компилятор предполагает наличие оператора PTR.
