# <a name="compiler-warning-level-1-c4143"></a>Предупреждение компилятора (уровень 1) C4143
pragma "same_seg" не поддерживается; используйте размещение __based  
  
 Директива **#Pragma same_seg** больше не поддерживается. Используйте [__based](../../cpp/based-pointers-cpp.md) ключевое слово вместо него.
