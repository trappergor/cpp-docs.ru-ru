# Предупреждение компилятора (уровень 1) C4679
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Компилятор обнаружил конструкцию, которая не поддерживается, не может быть импортирована из метаданных.  
  
 **"**   
 ***член* ": невозможно импортировать член**  
  
 Не пытайтесь использовать конструкцию.
