# <a name="compiler-warning-level-1-c4026"></a>Предупреждение компилятора (уровень 1) C4026
функция объявлена со списком формальных параметров  
  
 Объявление функции имеет формальные параметры, но определение функции не имеет. В последующих вызовах этой функции предполагается, что функция не принимает параметров.
