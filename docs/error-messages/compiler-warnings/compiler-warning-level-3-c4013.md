# <a name="compiler-warning-level-3-c4013"></a>Предупреждение компилятора (уровень 3) C4013

"функция" не определена; Предполагаемое внешнее возвращаемое целое число

Компилятор обнаружил вызов неопределенной функции.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Чтобы устранить ошибку, проверьте указанные ниже возможные причины ее возникновения.

1. Неправильное написание имени функции

1. Внешние функции, не имеющие прототипа**`extern`**
