# Предупреждение компилятора (уровень&#160;3) C4724
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

возможный остаток от деления на 0  
  
 Значение второго операнда в операции получения остатка от деления во время компиляции равно нулю, что привело к неопределенному результату.
