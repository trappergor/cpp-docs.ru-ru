# <a name="compiler-warning-level-1-c4603"></a>Предупреждение компилятора (уровень 1) C4603

"\<identifier >": макрос не определен или определение отличается после использования предкомпилированного заголовка

Макрос, определяемый заполнителем *идентификатор* , либо другой, либо больше не задан после использования предкомпилированного заголовка.
