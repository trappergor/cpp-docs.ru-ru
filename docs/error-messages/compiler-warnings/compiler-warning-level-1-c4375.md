# <a name="compiler-warning-level-1-c4375"></a>Предупреждение компилятора (уровень 1) C4375
неоткрытый метод «метод2» переопределяет «метод1»  
  
 Тип, реализующий другой тип, определенный методом переопределения, но переопределение не открытый. Таким образом метод не переопределяет метод базового типа.
