# Ошибка компилятора C2112
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"\-": для вычитания указателей требуется целый операнд или указатель  
  
 Совершена попытка выполнить вычитание указателей, указывающих на различные типы.
