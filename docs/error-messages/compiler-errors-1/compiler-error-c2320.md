# Ошибка компилятора C2320
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

требуется ":" после спецификатора доступа "спецификатор"  
  
 Ключевое слово `public`, `protected` или `private` должно сопровождаться двоеточием.
