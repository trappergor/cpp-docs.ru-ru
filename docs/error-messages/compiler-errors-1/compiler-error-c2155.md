# Ошибка компилятора C2155
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'?' недопустимый левый операнд, требуется арифметический тип или указатель  
  
 Выражение в левой части `?` нельзя сравнивать с нулем.  Следует использовать арифметический тип или выражение указателя, которое можно сравнивать с нулем.
