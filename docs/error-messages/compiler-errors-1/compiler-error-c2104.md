# <a name="compiler-error-c2104"></a>Ошибка компилятора C2104

"&" на учитывается битовым полем

Невозможно получить адрес битового поля.

Следующий пример приводит к возникновению ошибки C2104:

```
// C2104.cpp
struct X {
   int sb : 1;
};

int main() {
   X x;
   &x.sb;   // C2104
   x.sb;   // OK
}
```
