# <a name="fatal-error-c1351"></a>Неустранимая ошибка C1351

> Ошибка при загрузке dll "*dll*": несовместимая версия  

Найден неправильная версия библиотеки DLL.

Это указывает на проблему с установкой, и необходимо переустановить Visual Studio.
