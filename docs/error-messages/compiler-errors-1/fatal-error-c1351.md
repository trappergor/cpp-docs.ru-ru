# <a name="fatal-error-c1351"></a>Неустранимая ошибка C1351

> Ошибка при загрузке библиотеки DLL "*DLL*": несовместимая версия

Обнаружена неправильная версия библиотеки DLL.

Это указывает на проблему с установкой, и вам следует переустановить Visual Studio.
