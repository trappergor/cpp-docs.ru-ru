# <a name="compiler-error-c2020"></a>Ошибка компилятора C2020

«член»: переопределение члена «class»

Переопределяется члена, унаследованного от базового класса или структуры. Унаследованные члены не могут быть переопределены, если иное не объявлено как `virtual` в базовом классе.
