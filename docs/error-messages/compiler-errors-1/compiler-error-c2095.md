# <a name="compiler-error-c2095"></a>Ошибка компилятора C2095
«функция»: фактический параметр имеет тип «void»: параметр «число»  
  
 Параметр, переданный в функцию, тип `void`, который не является допустимым. Используйте указатель на void ( `void *`) вместо него.  
  
 `number` Указывает, какой параметр `void`.
