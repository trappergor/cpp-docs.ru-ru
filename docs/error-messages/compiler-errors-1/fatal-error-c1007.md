# Неустранимая ошибка C1007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

нераспознанная строка флага в параметре  
  
 Параметр командной строки содержит недопустимую строку. Убедитесь в отсутствии ошибок в переменной среды и командной строке **CL**.
