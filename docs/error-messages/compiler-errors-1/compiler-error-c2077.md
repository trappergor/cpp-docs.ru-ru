# <a name="compiler-error-c2077"></a>Ошибка компилятора C2077
инициализатор поля нескалярные «идентификатор»  
  
 Предпринята попытка инициализировать битовое поле с помощью нескалярного инициализатора (структуры, объединения, массива или класса). Используйте целое значение или значение с плавающей запятой.
