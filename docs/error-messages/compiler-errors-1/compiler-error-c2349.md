# Ошибка компилятора C2349
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"функция" не может быть скомпилирована как управляемая: "причина"; используйте прагма\-директиву \#pragma unmanaged  
  
 Дополнительные сведения см. в разделе [Предупреждение компилятора \(уровни 1 и 3\) C4793](../../error-messages/compiler-warnings/compiler-warning-level-1-and-3-c4793.md).
