# Неустранимая ошибка C1074
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

IDB — недопустимое расширение для PDB\-файла: "имя\_файла"  
  
 Компилятор ожидает, что файлы базы данных программы имеют расширение PDB.
