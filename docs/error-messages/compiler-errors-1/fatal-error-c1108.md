# <a name="fatal-error-c1108"></a>Неустранимая ошибка C1108

> не удается найти DLL: "*имя библиотеки dll*"

Указанную библиотеку DLL (*имя библиотеки dll*) не найден в пути.

Чтобы устранить эту ошибку, переустановите Visual C++ или скопируйте соответствующий DLL-файл из установки на компьютер.
