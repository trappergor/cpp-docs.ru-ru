# Ошибка компилятора C2309
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

для блока catch требуется объявление исключения в круглых скобках  
  
 Обработчик catch не является типом, требующим заключения в круглые скобки.  
  
 Следующий пример приводит к возникновению ошибки C2309:  
  
```  
// C2309.cpp  
// compile with: /EHsc  
#include <eh.h>  
class C {};  
int main() {  
   try {  
      throw "ooops!";  
   }  
   catch C {}   // C2309  
   // try the following line instead  
   // catch( C ) {}  
}  
```
