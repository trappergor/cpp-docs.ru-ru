# <a name="compiler-error-c2411"></a>Ошибка компилятора C2411

«Идентификатор»: Недопустимый член структуры или объединения в «контекст»

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Чтобы устранить ошибку, проверьте указанные ниже возможные причины ее возникновения.

1. Идентификатор не является членом отображаемой структуры или объединения в этом контексте.

1. Идентификатор не является членом структуры или объединения, указанных с помощью оператора выбора члена (.).
