# Ошибка компилятора C2338
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"идентификатор" поставщик атрибутов ATL  
  
 Атрибут ссылается на необъявленный идентификатор. Появится это сообщение об ошибке, связанное с ошибкой [Ошибка поставщика ATL ATL2010](http://msdn.microsoft.com/ru-ru/27a25924-b52b-4971-80d3-c3b3e9ffbe41).
