# <a name="compiler-error-c2315"></a>Ошибка компилятора C2315

"тип1": ссылка перехвачена "тип2" в строке "номер"

Тип исключения обрабатывается предыдущим обработчиком. Ссылка второго перехвата имеет тип первого перехвата.
