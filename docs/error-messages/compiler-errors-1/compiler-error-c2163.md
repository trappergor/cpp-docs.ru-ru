# <a name="compiler-error-c2163"></a>Ошибка компилятора C2163
«функция»: недоступна в качестве подставляемой функции  
  
 `intrinsic` Или `function` pragma перечислены функции, доступно не во встроенную форму. Например некоторые встроенные функции недоступны при компиляции программы с использованием параметра/CLR.
