# <a name="fatal-error-c1073"></a>Неустранимая ошибка C1073
Внутренняя ошибка при выполнении добавочной компиляции (файл компилятора «имя_файла», номер строки)  
  
 Перекомпилируйте файл без использования добавочной компиляции.
