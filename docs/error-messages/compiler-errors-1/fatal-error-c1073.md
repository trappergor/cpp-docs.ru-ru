# Неустранимая ошибка C1073
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Внутренняя ошибка при выполнении добавочной компиляции \(файл компилятора: "имя\_файла", строка: "номер"\)  
  
 Перекомпилируйте файл без использования добавочной компиляции.
