# <a name="compiler-error-c2151"></a>Ошибка компилятора C2151
несколько атрибутов языка  
  
 Функция имеет несколько ключевых слов ( `__cdecl`, `__stdcall`, или `__fastcall`) Указание соглашений о вызовах.
