# <a name="compiler-error-c2721"></a>Ошибка компилятора C2721
«спецификатор»: спецификатор класса хранения нельзя использовать между оператором и типом  
  
 Заданные пользователем преобразования типов применяются ко всем классам хранения, поэтому нельзя указать класс хранения в преобразовании типов.
