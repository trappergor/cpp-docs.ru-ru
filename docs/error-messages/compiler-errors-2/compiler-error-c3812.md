# <a name="compiler-error-c3812"></a>Ошибка компилятора C3812
«Свойство» должен быть первый токен в объявление свойства  
  
 При объявлении свойства, `__property` ключевое слово должно быть первой лексемой в строке.  
  
 C3812 доступен только с помощью параметра компилятора устаревшие **/CLR: oldSyntax**.
