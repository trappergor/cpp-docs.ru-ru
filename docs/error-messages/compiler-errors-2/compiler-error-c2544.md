# <a name="compiler-error-c2544"></a>Ошибка компилятора C2544
Ожидалось ")" для оператора «()»  
  
 Оператор вызова функции отсутствует левая круглая скобка.  
  
 Эта ошибка может быть вызвана расширения макроса.
