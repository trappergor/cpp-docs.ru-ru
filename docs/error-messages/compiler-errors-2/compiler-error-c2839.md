# Ошибка компилятора C2839
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

недопустимый возвращаемый тип «тип» для перегруженный оператор «\-\>»  
  
 Оператор `->` должен возвращать класс, структуру, объединение, или же ссылку на одну из этих сущностей.
