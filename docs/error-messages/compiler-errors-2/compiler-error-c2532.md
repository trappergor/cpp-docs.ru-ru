# Ошибка компилятора C2532
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"идентификатор": недопустимый модификатор для ссылки  
  
 Ссылка была изменена.  Ссылки нельзя изменять для указания на другой объект.  Вместо этого следует использовать указатель.
