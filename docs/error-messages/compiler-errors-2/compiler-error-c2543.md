# <a name="compiler-error-c2543"></a>Ошибка компилятора C2543
ожидается "]" для оператора «[]»  
  
 В операторе подписи пропущена отсутствует открывающая скобка.  
  
 Эта ошибка может быть вызвана расширения макроса.
