# Ошибка компилятора C2543
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

требуется "\]" для оператора "\[\]"  
  
 В операторе подписи пропущена левая скобка.  
  
 Эта ошибка может возникать при использовании расширения макроса.
