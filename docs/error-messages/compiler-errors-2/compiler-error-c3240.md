# <a name="compiler-error-c3240"></a>Ошибка компилятора C3240
«функция»: должна быть функцией-перегруженные абстрактный член «тип»  
  
 Базовый тип содержит функцию, которая была определена. Функция должна быть виртуальной.  
  
## <a name="example"></a>Пример  
 Следующий пример приводит к возникновению ошибки C3240.  
  
```  
// C3240.cpp  
// compile with: /c  
__interface I {  
   void f();  
};  
  
struct A1 : I {   
   void f() {}  
};  
  
struct A2 : I {   
   void f() = 0;  
};  
  
template <class T>   
struct A3 : T {  
   void T::f() {}  
};  
  
template <class T>   
struct A4 : T {  
   void T::f() {}  
};  
  
A3<A1> x;   // C3240  
A3<I> x2;  
A4<A2> x3;  
```
