# <a name="compiler-error-c2928"></a>Ошибка компилятора C2928

> явное создание экземпляра; "*идентификатор*" не является ни функцией, ни статическим элементом данных класса-шаблона "*Class-Name*"

Нельзя явно создать экземпляр члена *класса с именем* , который не является функцией или **`static`** переменной.
