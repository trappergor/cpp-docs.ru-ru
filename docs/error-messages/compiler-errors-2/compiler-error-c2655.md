# Ошибка компилятора C2655
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

идентификатор: недопустимое определение или повторное объявление в текущей области видимости  
  
 Идентификатор может быть переобъявляем только в глобальной области.  
  
 Следующий пример приводит к возникновению ошибки C2655:  
  
```  
// C2655.cpp  
class A {};  
class B {  
public:  
   static int i;  
};  
  
int B::i;  // OK  
  
int main() {  
   A B::i;  // C2655  
}  
```
