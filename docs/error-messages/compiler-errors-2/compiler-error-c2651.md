# <a name="compiler-error-c2651"></a>Ошибка компилятора C2651

«Тип данных»: слева от «operator» должен быть класс, структуру или объединение

Чтобы использовать параметр шаблона, если это класс, специализации шаблона класса с классом вместо целочисленного типа.
