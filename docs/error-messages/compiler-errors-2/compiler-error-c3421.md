# <a name="compiler-error-c3421"></a>Ошибка компилятора C3421
"тип": невозможно вызвать метод завершения для данного класса, поскольку он либо недоступен, либо не существует  
  
 Метод завершения является неявно закрытым, поэтому его нельзя вызвать за пределами включающего типа.  
  
 Дополнительные сведения см. в разделе [деструкторы и методы завершения в Практическое руководство: определение и использование классов и структур (C + +/ CLI)](../../dotnet/how-to-define-and-consume-classes-and-structs-cpp-cli.md#BKMK_Destructors_and_finalizers).  
  
## <a name="example"></a>Пример  
 Следующий пример приводит к возникновению ошибки C3421:  
  
```  
// C3421.cpp  
// compile with: /clr  
ref class A {};  
  
ref class B {   
   !B() {}  
  
public:  
   ~B() {}  
};  
  
int main() {  
   A a;  
   a.!A();   // C3421  
  
   B b;  
   b.!B();   // C3421  
}  
```
