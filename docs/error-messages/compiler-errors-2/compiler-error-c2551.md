# Ошибка компилятора C2551
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

для типа "void \*" требуется явное приведение  
  
 Указатель `void` присваивается непустому указателю путем неявного преобразования. Необходимо использовать явное приведение.
