# Ошибка компилятора C2962
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

синтаксическая ошибка: "токен": требуется определение функции\-члена класса\-шаблона до "}"  
  
 Этот токен является причиной синтаксической ошибки в объявлении шаблона.  
  
 Эта ошибка может возникать при несоответствии разделителей.
