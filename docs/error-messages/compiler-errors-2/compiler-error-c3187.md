# Ошибка компилятора C3187
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\_\_func\_\_' : можно использовать только в теле функции  
  
 Предварительно определенный макрос не доступен вне тела функции.  
  
 Чтобы исправить эту ошибку, переместите макрос в тело функции.
