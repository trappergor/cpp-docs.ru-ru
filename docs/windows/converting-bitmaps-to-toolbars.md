# <a name="converting-bitmaps-to-toolbars-c"></a>Преобразование растровых изображений в панели инструментов (C++)

Новая панель инструментов можно создать в проекте C++ путем преобразования растрового изображения. Рисунок из точечного рисунка преобразует изображениями кнопок панели инструментов. Обычно растровое изображение содержит несколько изображений кнопок отдельной с один образ для каждой кнопки. Образы могут быть любого размера; значение по умолчанию — 16 пикселей в ширину и высоту изображения. Можно указать размер изображения кнопок в [Создание ресурса панели инструментов-диалоговое окно](../windows/new-toolbar-resource-dialog-box.md) при выборе **редактор панелей инструментов** из **изображение** меню в редакторе изображений.

### <a name="to-convert-bitmaps-to-a-toolbar"></a>Чтобы преобразовывать растровые изображения на панель инструментов

1. Откройте существующий ресурс растрового изображения в [редактор изображений](../windows/image-editor-for-icons.md). (Если точечный рисунок уже не находится в RC-файл, щелкните правой кнопкой мыши RC-файл, выберите **импорта** в контекстном меню, перейдите к растровому изображению, вы хотите добавить в RC-файл, а затем нажмите кнопку **откройте**.)

2. Из **изображение** меню, выберите **редактор панелей инструментов**.

   [Создание ресурса панели инструментов-диалоговое окно](../windows/new-toolbar-resource-dialog-box.md) отображается. Можно изменить ширину и высоту изображения значков в соответствии с растрового изображения. Изображение панели инструментов отображается в редакторе панелей инструментов.

3. Чтобы завершить преобразование, измените команду **идентификатор** кнопки с помощью [окно "Свойства"](/visualstudio/ide/reference/properties-window). Введите новое **идентификатор** или выберите **идентификатор** из раскрывающегося списка.

   > [!TIP]
   > **Свойства** окно содержит кнопку канцелярской кнопки в заголовке окна. После нажатия этой кнопки включает или отключает **автоматическое скрытие** для окна. Чтобы быстро пролистать все свойства кнопок панели инструментов без необходимости повторно открыть windows отдельного свойства, включить **автоматическое скрытие** off поэтому **свойства** окно остается стационарным.

Можно также изменить идентификаторы команд кнопок на новой панели инструментов с помощью [окно "Свойства"](/visualstudio/ide/reference/properties-window). Сведения о редактировании новой панели инструментов см. в разделе [создание, перемещение и редактирование кнопок панели инструментов](../windows/creating-moving-and-editing-toolbar-buttons.md).

Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в приложениях для настольных систем](/dotnet/framework/resources/index) в *руководства разработчика .NET Framework*. Сведения о вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам, см. в разделе [Создание файлов ресурсов для приложений рабочего стола](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Сведения о глобализации и локализации ресурсов в управляемых приложениях, см. в разделе [Globalizing and Localizing .NET Framework Applications](/dotnet/standard/globalization-localization/index).

## <a name="requirements"></a>Требования

MFC или ATL
