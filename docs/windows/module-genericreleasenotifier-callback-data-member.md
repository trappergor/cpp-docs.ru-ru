# <a name="modulegenericreleasenotifiercallback-data-member"></a>Элемент данных Module::GenericReleaseNotifier::callback_

## <a name="syntax"></a>Синтаксис  
  
```  
T callback_;  
```

 **Пространство имен:** Microsoft::WRL
