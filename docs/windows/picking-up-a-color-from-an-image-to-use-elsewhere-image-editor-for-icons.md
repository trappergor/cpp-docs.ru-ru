# <a name="picking-up-a-color-from-an-image-to-use-elsewhere-image-editor-for-icons"></a>Выбор на изображении образца цвета для использования в другом фрагменте (редактор изображений для значков)
**Выбор цвета**, или цвет pickup, средство выполняет любой цвет на изображении, текущий цвет переднего плана или фонового цвета ли вы левой или правой кнопкой мыши. Чтобы отменить **Выбор цвета** инструмент, выберите любой другой инструмент.  
  
> [!TIP]
>  Всплывающие подсказки отображаются при наведении курсора на кнопку панели инструментов. Эти советы помогут определить назначение каждой кнопки.  
  
### <a name="to-pick-up-a-color"></a>Для выбора цвета  
  
1.  На **редактор изображений** панели инструментов (или из **изображение** меню **средства** команды), нажмите кнопку **Выбор цвета** средство.  
  
2.  Выберите цвет, который вы хотите взять из образа.  
  
    > [!NOTE]
    >  После его выбора цвета, редактор изображений повторно активирует средство недавно использованных.  
  
3.  Рисование с помощью левой кнопки мыши для цвет переднего плана, или правой кнопкой мыши для цвета фона.

Сведения о вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам, см. в разделе [Создание файлов ресурсов для приложений рабочего стола](/dotnet/framework/resources/creating-resource-files-for-desktop-apps).
