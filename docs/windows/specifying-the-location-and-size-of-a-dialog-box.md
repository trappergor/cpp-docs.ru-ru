# <a name="specifying-the-location-and-size-of-a-dialog-box-c"></a>Задание местоположения и размера диалогового окна (C++)

Расположение и размер в диалоговом окне C++, а также расположение и размер элементов управления, измеряются в диалоговых единицах. Значения для отдельных элементов управления и диалоговом окне отображаются в правом нижнем углу строки при их выборе состояния Visual Studio.

Существует три свойства, которые можно установить в [окно "Свойства"](/visualstudio/ide/reference/properties-window) для указания, где диалоговое окно будет отображаться на экране. **Center** свойство является логическое значение; Если значение равно **True**, диалоговое окно всегда будет отображаться в центре экрана. Если ему присвоено **False**, затем можно задать **XPos** и **YPos** свойства, чтобы явно определить, когда на экране появится диалоговое окно. Позиция свойства, значения смещения в верхнем левом углу области просмотра, который определен как `{X=0, Y=0}`. Позиция также основана на **абсолютное выравнивание** свойство: Если **True**, координаты указываются относительно экрана; Если **False**, координаты указываются относительно диалогового окна окно владельца.

Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в приложениях для настольных систем](/dotnet/framework/resources/index) в *руководства разработчика .NET Framework*. Сведения о вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам, см. в разделе [Создание файлов ресурсов для приложений рабочего стола](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Сведения о глобализации и локализации ресурсов в управляемых приложениях, см. в разделе [Globalizing and Localizing .NET Framework Applications](/dotnet/standard/globalization-localization/index).
