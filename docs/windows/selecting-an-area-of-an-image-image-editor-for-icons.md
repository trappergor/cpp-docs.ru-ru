# <a name="selecting-an-area-of-an-image-image-editor-for-icons"></a>Выбор области изображения (редактор изображений для значков)

Инструменты выбора можно использовать для определения области изображения, которое требуется очистить, Вырезать, копировать, изменение размера, обратить или перемещение. С помощью **Прямоугольное выделение** средство можно определить и выбрать прямоугольную область изображения. С помощью **произвольное выделение** инструмент, можно нарисовать Рисованная линия структуры области, который требуется вырезать, копировать и другие операции.

> [!NOTE]
> См. в разделе **Прямоугольное выделение** и **произвольное выделение** средства, изображенного на [панель инструментов редактора изображений](../windows/toolbar-image-editor-for-icons.md) или просмотреть всплывающие подсказки, связанный с каждой кнопки на **Редактор изображений** панели инструментов.

Можно также создать настраиваемую кисть из выделенной области. Дополнительные сведения см. в разделе [Создание кисти Custom](../windows/creating-a-custom-brush-image-editor-for-icons.md).

### <a name="to-select-an-area-of-an-image"></a>Чтобы выбрать область изображения

1. На **редактор изображений** панели инструментов (или из **изображение** меню **средства** команды), щелкните средство выбора, вы хотите.

2. Переместите курсор на угол области изображения, которое вы хотите выбрать. Перекрестие отображаются, когда курсор находится над изображением.

3. Перетащите курсор противоположного угла области, которую вы хотите выбрать. Прямоугольник показывает, будут выбраны какие пикселей. Все точки внутри прямоугольника, включая расположенные в прямоугольник, включаются в выделении.

4. Отпустите кнопку мыши. Границы выделения заключает выбранной области.

### <a name="to-select-an-entire-image"></a>Чтобы выбрать всего изображения

1. Щелкните изображение, за пределами текущего выделения. Границы выделения меняет фокус и охватывает все изображения, еще раз.

Сведения о вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам, см. в разделе [Создание файлов ресурсов для приложений рабочего стола](/dotnet/framework/resources/creating-resource-files-for-desktop-apps).
