# <a name="hstringhstring-destructor"></a>Деструктор HString::~HString

## <a name="syntax"></a>Синтаксис  
  
```cpp  
~HString() throw()  
```  
  
## <a name="requirements"></a>Требования

 **Пространство имен:** Microsoft::wrl:: wrappers
