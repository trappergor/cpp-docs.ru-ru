# <a name="reverse-stlclr"></a>reverse (STL/CLR)
Изменяет порядок элементов в диапазоне на обратный.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
template<class _BidIt> inline  
    void reverse(_BidIt _First, _BidIt _Last);  
```  
  
## <a name="remarks"></a>Примечания  
 Эта функция работает так же, как функция стандартной библиотеки C++ `reverse`. Дополнительные сведения см. в разделе [обратного](../standard-library/algorithm-functions.md#reverse).  
  
## <a name="requirements"></a>Требования  
 **Заголовок:** \<cliext/алгоритм >  
  
 **Пространство имен:** cliext
