# An invalid cookie was passed to UnregisterExtenderProvider().
Указан файл cookie, который больше не является допустимым.  
  
### Чтобы исправить эту ошибку  
  
1.  Верните то же самое значение cookie, которое было получено при вызове RegisterExtenderProvider\(\).
