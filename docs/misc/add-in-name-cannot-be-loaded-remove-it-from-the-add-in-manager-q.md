# Add-in &lt;name&gt; cannot be loaded. Remove it from the Add-in Manager?
Не удается найти выбранную надстройку.  Эта ошибка обычно возникает, когда для выбранной надстройки отсутствует DLL или другой необходимый компонент.  
  
### Чтобы исправить эту ошибку  
  
1.  Переустановите надстройку.
