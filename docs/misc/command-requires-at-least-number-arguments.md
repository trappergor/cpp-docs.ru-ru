# Command requires at least &lt;number&gt; arguments.
Эта ошибка обычно возникает, если для команды было введено слишком мало аргументов.  
  
### Чтобы исправить эту ошибку  
  
1.  Проверьте в документации правильность синтаксиса команды и повторите попытку.
