# This project cannot be added to the solution because it already exists in the solution in a hidden Solution Folder.
Проект может существовать в решении, находясь только в одном расположении.  
  
### Чтобы исправить эту ошибку  
  
1.  Чтобы отобразить скрытые папки решения, выберите решение или папку решения, содержащую скрытую папку решения, а затем в меню **Проект** выберите пункт **Отобразить папки**.
