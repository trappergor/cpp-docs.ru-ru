# Ошибка сборки проекта PRJ0011
Не удалось создать канал, необходимый для запуска построения. Не удается создать событие, необходимое для запуска сборки.  
  
 Если ресурсы системы находятся на низком уровне, выполните указанные ниже действия.  
  
-   Закройте некоторые выполняющиеся приложения.  
  
-   Перезапустите Visual Studio, чтобы освободить ресурсы.  
  
-   Перезагрузите систему.
