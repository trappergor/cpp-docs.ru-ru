# Разрешение вопросов, связанных с исключениями: System.Workflow.Activities.WorkflowAuthorizationException
Исключение <xref:System.Workflow.Activities.WorkflowAuthorizationException> вызывается при сбое проверки роли из–за отсутствия указанного идентификатора в <xref:System.Workflow.Activities.WorkflowRoleCollection>.
