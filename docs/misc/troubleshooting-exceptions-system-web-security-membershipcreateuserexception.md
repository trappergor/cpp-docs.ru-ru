# Разрешение вопросов, связанных с исключениями: System.Web.Security.MembershipCreateUserException
Это исключение вызывается, когда поставщик членства не смог создать пользователя.  
  
## Примечания  
 Свойство <xref:System.Web.Security.MembershipCreateUserException.StatusCode%2A> содержит причину данного исключения.
