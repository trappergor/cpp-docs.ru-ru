# Command was not specified.
Эта ошибка возникает, если в окне **Команда** или поле **Найти\/Команда** введена пустая строка.  
  
### Чтобы исправить эту ошибку  
  
1.  Введите допустимую команду.  Для получения дополнительной информации см. [Команды Visual Studio](../Topic/Visual%20Studio%20Commands.md).
