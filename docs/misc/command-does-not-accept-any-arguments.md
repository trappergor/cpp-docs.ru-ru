# Command does not accept any arguments.
Эта ошибка обычно возникает, если для команды, которая принимает только параметры, был введен аргумент.  
  
### Чтобы исправить эту ошибку  
  
1.  Проверьте в документации правильность синтаксиса команды и повторите попытку.
