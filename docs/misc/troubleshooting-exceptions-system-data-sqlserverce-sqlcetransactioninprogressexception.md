# Разрешение вопросов, связанных с исключениями: System.Data.SqlServerCe.SqlCeTransactionInProgressException
Исключение `System.Data.SqlServerCe.SqlCeTransactionInProgressException` вызывается при попытке приложения изменить базу данных во время выполнения другой транзакции. Для получения дополнительной информации см. [Библиотека классов SQL Server Compact 3.5](http://go.microsoft.com/fwlink/?LinkID=102595).
