# Cannot find main menu commandbar in resources.
Эта ошибка возникает, когда интегрированной среде разработки при запуске не удается найти командную строку MenuBar.  DLL\-файл удален, перемещен или поврежден.  
  
### Чтобы исправить эту ошибку  
  
-   Для восстановления или переустановки продукта воспользуйтесь средством **Установка и удаление программ**.
