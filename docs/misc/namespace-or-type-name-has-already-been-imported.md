# Пространство имен или тип &quot;&lt;имя&gt;&quot; уже импортированы
Предпринята попытка импортировать пространство имен более одного раза.  
  
 **Идентификатор ошибки:** BC31051  
  
### Исправление ошибки  
  
-   Удалите лишние операторы `Imports`.
