# <a name="fmod-function"></a>Функция fmod
**ANSI 4.5.6.4** Действие в случае, если второй аргумент функции `fmod` равен нулю: возникает ошибка домена или возвращается нулевое значение  
  
 Если второй аргумент функции `fmod` равен нулю, функция возвращает ноль.
