 **[ ]   ( )   .   ->++   --    &   \*   +   -   ~   !   sizeof/   %   <\<   >>   <>   \<=   >=   ==   !=   ^   &#124;   &&   !!?   :=   \*=   /=   %=   +=   -=   <\<=   >>=   &=   ^=   &#124;=,   #   ##**

 **=   \*=   /=   %=   +=   -=   <\<=   >>= &=   ^=   &#124;=**
