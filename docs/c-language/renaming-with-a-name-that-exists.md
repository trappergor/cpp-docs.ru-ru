# <a name="renaming-with-a-name-that-exists"></a>Переименование с использованием существующего имени

**ANSI 4.9.4.2** Что происходит, если файл с новым именем, указанным в вызове функции **rename**, уже существует

При попытке переименовать файл с использованием уже существующего имени функция **rename** завершается неудачей и возвращает код ошибки.
