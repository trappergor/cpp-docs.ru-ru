# Остатки
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.3.5** Знак остатка в целочисленном отделении  
  
 Знак остатка совпадает со знаком делимого.  Например:  
  
```  
 50 / -6 == -8  
 50 % -6 ==  2  
-50 /  6 == -8  
-50 %  6 == -2  
```
