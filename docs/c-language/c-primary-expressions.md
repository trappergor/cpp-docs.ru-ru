# <a name="c-primary-expressions"></a>Первичные выражения в C

Операнды в выражениях называются основными выражениями.

## <a name="syntax"></a>Синтаксис

*primary-expression*: *identifier*

*constant*

*string-literal*

**(**  *expression*  **)**

*expression*: *assignment-expression*

*expression*  **,**  *assignment-expression*
