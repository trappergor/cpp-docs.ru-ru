# Количество битов на символ
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 2.2.4.2 .1** Число битов в символе  
  
 Количество битов в символе представляется константой **CHAR\_BIT** в манифесте.  В файле LIMITS.H для константы **CHAR\_BIT** задано значение 8.
