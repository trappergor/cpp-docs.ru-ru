# <a name="file-buffering"></a>Буферизации файла
**ANSI 4.9.3** Характеристики буферизации файлов  
  
 Дисковые файлы, полученные через стандартные функции ввода-вывода, буферизуются полностью. По умолчанию буфер удерживает 512 байтов.
