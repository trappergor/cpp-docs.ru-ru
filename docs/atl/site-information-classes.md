# <a name="site-information-classes"></a>Классы сведения веб-узла
Следующие классы позволяют объектам взаимодействия со своим сайтом:  
  
-   [Интерфейс IObjectWithSiteImpl](../atl/reference/iobjectwithsiteimpl-class.md) получает и задает указатель объекта сайта. Используется для объектов, которые не являются элементами управления.  
  
-   [IOleObjectImpl](../atl/reference/ioleobjectimpl-class.md) получает и задает указатель объекта сайта. Используется для элементов управления.
